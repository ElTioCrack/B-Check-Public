# B-Check 💵

**Validador de Billetes Bolivianos** | App para identificar billetes del avión de El Alto

🌐 **Página oficial:** [bchecverify.netlify.app](https://bchecverify.netlify.app)

[![Flutter Version](https://img.shields.io/badge/Flutter-3.0+-blue.svg)](https://flutter.dev)
[![Android](https://img.shields.io/badge/Android-5.0%2B-brightgreen.svg)](https://www.android.com)

## ✨ Características

- 📱 **Detección manual** - Ingresa el número de serie manualmente
- 📸 **Escáner simple** - Escanea un billete a la vez
- 🔄 **Escáner múltiple** - Escanea varios billetes simultáneamente
- 🔊 **Accesibilidad** - Compatible con lectores de pantalla
- 📶 **Offline first** - Funciona sin internet

[![Download](https://img.shields.io/badge/Download-APK-blue.svg)](https://github.com/ElTioCrack/B-Check-Public/releases/download/v1.1.0/B-Check_v1.1.0.apk)

**Última versión:** `v1.1.0` (Android 5.0+)

## 🚀 Cómo usar

1. Selecciona el corte del billete (10, 20 o 50 Bs)
2. Elige el modo de escaneo (manual, simple o múltiple)
3. Escanea o ingresa el número de serie
4. ¡Listo! La app te dirá si es válido o inválido
